魔法元素是魔法粒子的基本构成
各种魔法元素的不同组合会生成新的物质
基础元素:
风元素
地元素
火元素
水元素
光元素
暗元素
空间元素
时间元素

其中需要注意的是 光元素 和 暗元素
光元素并不是 光亮 的意思, 而是代表任何物质的一个 `正面`, 某些地区也会将光元素成为 阳元素
相反的, 暗元素就是物质的 `反面`, 某些地区也会将光元素成为 阴元素

魔法公式:
(+) :: 元素 -> 元素 -> 元素
上面这条式子意思是, 两个元素相融合, 会生成另一种元素        // Haskell 乱入
比如
光 + 暗 = 平衡
如果是两种相同的元素, 则一般不发生反应
如果有两种以上的元素进行合成, 那么部分需要先合成的元素, 可以用括号括起来
比如
(火 + 水) + (光 + 暗) = 相性
当然也可以化简为
混沌 + 平衡 = 相性

基础元素与其他元素相融合的时候不需要添加 `融合药剂`
而非基础元素相融合的时候需要 `融合药剂` 来完成融合的过程

有的魔法公式的元素前面会有若干个感叹号(!), 表示这种元素的数量需要很多, 最多为9个感叹号, 不过这种情况一般不多
9个感叹号的元素量, 即使是同种元素相加, 也会使其发生 质变

常用的魔法公式:
风 + 地 = 传播
风 + 火 = 蔓延
风 + 水 = 涡旋
风 + 光 = 冷
风 + 暗 = 暴风

地 + 火 = 金
地 + 水 = 生命
地 + 光 = 幸运
地 + 暗 = 污秽

火 + 水 = 相克(或叫做混沌)
火 + 光 = 光明
火 + 暗 = 地狱

水 + 光 = 反射
水 + 暗 = 毒

光 + 暗 = 平衡

时间元素 + 空间元素 = 奇点
奇点 + !!!!!!!!!风 + !!!!!!!!!地 + !!!!!!!!!火 + !!!!!!!!!水 + !!!!!!!!!光 + !!!!!!!!!暗 = 世界 // 危险

关于感叹号(!)
n个感叹号表示需要 (0 < n && n <= 9)
2^n^n 倍的元素(x^y 表示 x的y次方)
当感叹号达到 9 的时候, 需要的元素量是非常多的

基础元素, 非基础元素与混合元素
基础元素即为上面的 8 种基础元素
而非基础元素表示 除基础元素外的其他元素
混合元素表示 两种没有相对关系的元素的融合
什么是相对关系呢
例如 平衡 和 相克, 平衡元素 是将元素往好的方向平衡, 相克元素 是将元素往坏的方向混沌
平衡 平衡了混沌, 而混沌又 混沌了平衡
这两者最终走向同一个重点, 就是 中和(相性)
